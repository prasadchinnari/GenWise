
# GenWise — Productivity & EduScope Integrated (Final)

## Overview
GenWise is a student-focused web app combining productivity tools (tasks, focus timer) with career guidance (EduScope module). This integrated version is responsive and ready for GitHub Pages or Firebase Hosting.

## Features
- Tasks with localStorage persistence
- Focus timer (25-minute default)
- Motivational quotes
- EduScope: Interest Quiz, Career Roadmaps
- College Finder (sample data)
- Theme toggle
- Firebase placeholder for future integration

## How to run
1. Extract the ZIP and open `index.html` in your browser.
2. Use productivity tools and the EduScope module.
3. To persist data across devices, integrate Firebase and replace `assets/js/firebase-config.js` with your project's config.
